# SlideShowApp

## Man kör projektet genom:
1. installera dependencies med:
```
 npm install
```
2. Starta projektet:
```
 npm start
```

## Använda RN-Komponenter

- Image
- FlatList
- Button
- Text
- View
- TouchableOpacity
- StyleSheet
- SafeAreaProvider
- MapView,
- Marker
- PROVIDER_GOOGLE
- Dimensions

## Använda Expo-komponenter

- ImagePicker
- StatusBar
- MaterialCommunityIcons
- SecureStore
- Device
- AV


## Övriga använda komponenter

- Formik
- Yup
- Nanoid
